# 💰 EMIPredict_AI

A machine learning web app that predicts **Equated Monthly Installments (EMI)** based on customer financial and demographic data.  
Built using **Python**, **Streamlit**, and **MLflow** for model tracking.

---

## 🚀 Features

- 🧹 **Data Cleaning & Feature Engineering**  
- 📊 **Exploratory Data Analysis (EDA)** using Pandas & Plotly  
- 🤖 **Model Training & Evaluation** with Scikit-learn  
- 🧠 **MLflow Tracking** for model versions  
- 🌐 **Interactive Web App** using Streamlit  
- 📈 **EMI Prediction** from user inputs

---
